# Privacy Policy

TrustLens is built on **zero-surveillance principles**.

---

## Data Collection

TrustLens does NOT:
- Store user input permanently
- Track users across sessions
- Use cookies for profiling
- Sell or share data

---

## Input Handling

- Text is processed transiently
- Analysis occurs in-memory
- No personal identifiers are required

---

## Third-Party APIs

Optional integrations (e.g., fact-check databases) are:
- Queried anonymously
- Never receive user identifiers
- Subject to strict rate limits

---

## Analytics

If analytics are used:
- They are aggregated
- They are non-identifying
- They measure system health, not users

---

## User Control

Users may:
- Use TrustLens without accounts
- Clear results instantly
- Avoid all tracking mechanisms

---

## Compliance

TrustLens is aligned with:
- GDPR principles
- Data minimization standards
- Privacy-by-design frameworks
